Introduction to HTML:


HTML, or HyperText Markup Language, is the backbone of web development. It provides the structure and markup for creating content on the internet. HTML uses a system of tags to define various elements within a web page.


Key Concepts:


HyperText Markup Language:
HTML is not a programming language; it's a markup language used to structure content on the web. It uses a system of tags to define different elements.
Web Page Structure:
HTML is used to structure a web page by dividing it into sections like headers, paragraphs, images, and more. The structure created with HTML forms the basis for styling and functionality


Example:

    
      <!DOCTYPE html>
      <html lang="en">
      <head>
        <title>My First HTML Page</title>
      </head>
      <body>
        <h1>Welcome to My Website</h1>
        <p>This is a simple introduction to HTML.</p>
      </body>
      </html>

Explanation:

    
     <!DOCTYPE html declares the HTML version.>
     <html is the root element of the HTML document.>
     <head contains meta-information about the HTML document.>
     <body contains the content of the HTML document.>
     <h1, p are HTML tags for heading and paragraph respectively.>

HTML Document Structure:


The HTML document structure is the foundation of web development and consists of key elements that organize and define the content of a webpage.


*The document typically starts with*


<!DOCTYPE html: This declaration at the beginning tells the browser the version of HTML being used (HTML5 in this case).>
<html: is the root element, containing the entire document.>
<head: includes meta-information like character set, title, and linked stylesheets.>
<body: encloses the main content of the webpage.>

HTML Elements and Tags:


HTML elements and tags are fundamental building blocks used to structure and define content in a web page.


HTML Element:


An HTML element is a fundamental component of an HTML document. It consists of an opening tag, content, and a closing tag.
Example: <p>This is a paragraph.</p>
